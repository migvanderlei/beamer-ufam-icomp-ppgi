Prévia

Exemplos da identidade visual do tema: capa, divisor de seção e slide de agradecimentos.

Como utilizar

A forma recomendada de uso é pelo Overleaf. A compilação local permanece disponível para quem preferir trabalhar offline.

  1. Crie um novo projeto no Overleaf e envie beamerthemeIComp.sty, example.tex e a pasta logos/. Em planos com sincronização GitHub, também é possível importar o repositório diretamente (New Project → Import from GitHub).
  2. Em Menu → Main document, selecione example.tex e preencha o bloco METADADOS com título, autor e demais informações.
  3. Utilize o compilador pdfLaTeX e compile o documento duas vezes, a fim de atualizar o sumário e a numeração total de slides.
  1. Clone o repositório e acesse o diretório do projeto.
  2. Edite example.tex, preenchendo o bloco METADADOS (título, autor, orientador, contato etc.).
  3. Compile com make (preferencialmente via latexmk, que executa as passagens necessárias): 
git clone https://github.com/migvanderlei/beamer-ufam-icomp-ppgi.git
cd beamer-ufam-icomp-ppgi
make          # gera example.pdf
make all      # gera example.pdf e main.pdf
