Prévia

Capa, divisor de seção e slide de agradecimentos — a identidade visual do tema.

Começar em 3 passos

  1. No Overleaf, crie um projeto e envie beamerthemeIComp.sty, example.tex e a pasta logos/.
  2. Em Menu → Main document, escolha example.tex. Preencha o bloco METADADOS.
  3. Compilador: pdfLaTeX. Compile duas vezes (sumário e total de slides).
  1. Clone o repositório e entre na pasta do projeto.
  2. Edite example.tex (bloco METADADOS) com título, autor e contato.
  3. Compile com pdfLaTeX duas vezes:
git clone https://github.com/migvanderlei/beamer-ufam-icomp-ppgi.git
cd beamer-ufam-icomp-ppgi
pdflatex example.tex
