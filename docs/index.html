Apresentações com a cara do IComp

Tema Beamer em teal + lima, pronto para Overleaf e pdfLaTeX. Starter em português, strings padrão via babel.

Ver no GitHub Baixar example.pdf

Prévia

Capa, divisor de seção e slide de agradecimentos — a identidade visual do tema.

Começar em 3 passos

  1. Clone o repositório ou envie beamerthemeIComp.sty, example.tex e logos/ para o Overleaf.
  2. Preencha o bloco METADADOS em example.tex.
  3. Compile com pdfLaTeX duas vezes.
